# Versed's documentation

Versed's documentation is split into several parts:

- [The README](../README.md)
  — Contains installation instructions and a brief example.
- [The Versed schema language](language.md)
  — A detailed description of the schema language.
- [The Versed compiler usage guide](usage.md)
  — A brief description of the commands supported by `versed`.
- [The thesis written about Versed](https://dspace.cuni.cz/browse?type=author&value=Swart,%20Benjamin&locale-attribute=en)
  — Contains a detailed description of the language (Chapter 4),
  of interactive migrations (Section 5.2),
  the generated migration functions (Section 6.3),
  many examples for interactive migrations (Chapter 7),
  and much more.
- [The developer's guide](development.md)
  — A short overview of Versed's architecture
  and instructions on how to build the source code documentation.
